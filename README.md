## 2. Getting Started

### Prerequisites

* Node.js (version 18 or higher)
* npm (Node Package Manager, usually comes with Node.js)
* Cypress
* Github

### Installation

#### 1. Clone the repository
```bash
git https://github.com/Dolores-96/Aestus/tree/main
cd parra-tests
```

#### 2. Run Cypress tests
```bash
npx cypress open 
```
